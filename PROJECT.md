# Проект: Landing Page — Ремонт квартир

## Описание

Демонстрационный одностраничный сайт для компании по ремонту квартир под ключ. Создан как кейс для портфолио веб-разработчика.

Целевая аудитория: владельцы квартир в Москве и области, планирующие ремонт (косметический, капитальный, евроремонт).

## Технологии

- **HTML5** — семантическая разметка
- **CSS3** — встроенные стили
- **JavaScript** — простой калькулятор стоимости (встроенный)
- **Адаптивный дизайн** — mobile-first
- **Нет внешних зависимостей** — всё в одном файле

## Структура страницы

| Секция | Что внутри |
|--------|-----------|
| Header | Логотип РемСтрой, навигация, CTA «Бесплатный замер» |
| Hero | H1, подзаголовок, цена от 4 500₽/м², прогресс-бары с этапами |
| Виды ремонта | 3 карточки: Косметический, Капитальный (популярный), Евроремонт |
| Процесс | 5 шагов: замер, смета, договор, ремонт, сдача |
| Калькулятор | Интерактивный JS-калькулятор: площадь × тип × комнаты = итог |
| Контакты | Контакты + форма заявки на бесплатный замер |
| Footer | Логотип, ссылки, копирайт |

## Особенности

### Калькулятор (JavaScript)

Встроен в страницу. Формула:
```
Итог = площадь × цена_за_м² × коэффициент_комнат
```

Где:
- Косметический: 4 500 ₽/м²
- Капитальный: 8 500 ₽/м²
- Евроремонт: 15 000 ₽/м²
- Коэффициент комнат: 1.0–1.15

Код калькулятора находится в `<script>` внизу страницы.

## Как редактировать

### Сменить цены

Найди секцию `#types` и обнови цифры в `.price`. Обнови также опции `<select>` в калькуляторе (в `<script>` внизу).

### Сменить этапы ремонта

Найди `.steps` в секции `#process`. Каждый `.step` — один этап. Можно добавить/убрать шаги.

### Добавить фото

Замени CSS-градиенты на `background-image`. Добавь папку `images/` и пропиши пути.

### Изменить формулу калькулятора

Найди `<script>` внизу файла и поменяй логику в функции `calculate()`.
